# Assets

This directory should contain the following image files:

- **icon.png** - App icon (1024x1024px)
- **adaptive-icon.png** - Android adaptive icon (1024x1024px)
- **splash.png** - Splash screen image (1284x2778px)
- **favicon.png** - Web favicon (48x48px)

## Placeholder Assets

For development purposes, you can use placeholder images or generate them using tools like:
- https://www.figma.com
- https://www.canva.com
- Expo's asset generator

## Design Guidelines

All assets should follow the Todoist color scheme:
- Primary color: #de4c4a (red)
- Background: #1f1f1f (dark gray)
- Use simple, minimalist designs
